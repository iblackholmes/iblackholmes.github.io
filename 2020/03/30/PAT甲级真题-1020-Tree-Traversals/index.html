题目描述

Description

Suppose that all the keys in a binary tree are distinct positive integers. Given the postorder and inorder traversal sequences, you are supposed to output the level order traversal sequence of the corresponding binary tree.

Input Specification

Each input file contains one test case. For each case, the first line gives a positive integer $N$ ($\leq$30), the total number of nodes in the binary tree. The second line gives the postorder sequence and the third line gives the inorder sequence. All the numbers in a line are separated by a space.

Output Specification

For each test case, print in one line the level order traversal sequence of the corresponding binary tree. All the numbers in a line must be separated by exactly one space, and there must be no extra space at the end of the line.

分析与思路

题目大意

假设树的键值唯一，给出树的后序遍历和中序遍历，输出树的层次遍历。(后序中序转层序)

类似题型：

  1. LeetCode P105 Construct Binary Tree from Preorder and Inorder Traversal
  2. LeetCode P106 Construct Binary Tree from Inorder and Postorder Traversal

分析

最基本的方法——划分法

已知树的后序遍历为LR⭕，中序遍历为L⭕R。因此可以先通过后序遍历序列确定当前子树根节点的value，再找到根节点value位于中序遍历序列的位置，由此可以推断出左右子树的长度，递归即可求解。

这种方法有两个方向可以优化

  1. 再递归构造树前，先用一个map存储中序遍历序列，这样在确定根节点value位于中序遍历序列的位置时不用再循环一遍查找
  2. 不必造树[^2]，给node新增一个index属性，记录其在层次遍历中的位置，在递归遍历过程中更新index值即可，结束便利后排序即为中序遍历序列

删点法——O(n)[^1]

实际上是以⭕RL方式遍历构造树

对于前序中序造树，则是以⭕LR方式遍历构造树

需要注意的是，递归构造前，需要给中序遍历序列头部添加一个哨兵元素(对于前序中序造树，在中序遍历序列尾部添加)
